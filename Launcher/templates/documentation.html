Documentation

Prerequisite

  • Log in or Register with the system first
  • Apply for a Role to the Administration to manage projects: Settings -> My Profile -> Request to Administrator for Roles

Deploy

  1. Add Server Machine if not added: Host Machines -> Add New Host
  2. Add Project information: Projects -> Create New Project
  3. Deploy Project: Projects -> Deploy a Project -> Select which project to deploy

Special

  1. Add new Office Product / View available products: Products
  2. Execute single commands or tasks: Tasks
